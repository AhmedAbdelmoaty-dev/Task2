# Task

## Features

- **Clean Architecture**:  API, Application, Domain, and Infrastructure layers  
- **CQRS pattern** with **MediatR**: clear separation of commands (writes) and queries (reads)  
- **PostgreSQL** as the database  
- **Redis caching** for optimized read performance  
- **Result pattern** for consistent API responses  
- **FluentValidation** for request validation  
- **Global exception handling** via middleware

- **C#**

- **PostgreSQL**  
- **Redis**  
- **MediatR**
